迁移日志

背景

在某个清晨，心血来潮想写个文档，在我使用hexo生成文档时发现我的 hexo 已经坏了，由于我使用的是 archlinux，我每天都有升级的习惯，奶奶的 hexo 每次都要为包操心。

方法

方法还是比较简单的，只需要更改hexo的日期格式和toml格式的标签和分类就行，具体网上的教程已经很多了就不赘述了。

对比

优点

  • 由于hugo是go语言编写，对比hexo提升了几十倍的编译速度。

  • 再也不需要维护那么的 npm 包了，迁移变的更简单。

缺点

  • 对比hexo主题实在是太丑了，我再也没有花里胡哨的博客了。

  • 我暂时也没有找到 hugo 嵌入看板娘的方法，我不想去维护 npm 包了。
