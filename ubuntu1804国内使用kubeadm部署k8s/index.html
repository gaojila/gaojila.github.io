安装前准备

kvm 准备 3 台虚拟机

  • kube-master
  • kube-node1
  • kube-node2

关闭防火墙

1

sudo ufw disable

关闭 swap

  • 临时关闭
1

sudo swapoff -a
  • 永久关闭
1

/etc/fstab 注释swap行

安装 docker

卸载旧 docker

1

sudo apt-get remove docker docker-engine docker.io

彻底删除 docker-ce 和 docker-cli

1

sudo apt-get autoremove docker-ce

安装依赖

1
2
3
4
5

sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    software-properties-common

添加 docker 的 gpg key

为了使用 https 访问，使用阿里云加速

1

curl -fsSL https://mirrors.aliyun.com/docker-ce/linux/ubuntu/gpg | sudo apt-key add -

设置 docker 镜像源

1
2
3
4

sudo add-apt-repository \
   "deb [arch=amd64] https://mirrors.aliyun.com/docker-ce/linux/ubuntu \
   $(lsb_release -cs) \
   stable"

更新源

1

sudo apt update && sudo apt-get update

安装 docker 18.06

查看 docker-ce 版本

1

apt-cache madison docker-ce

安装指定版本

1

sudo apt-get install -y docker-ce=18.06.3~ce~3-0~ubuntu

默认启动

1

sudo systemctl enable docker && sudo systemctl start docker

修改 docker 启动参数，进行加速

sudo systemctl restart docker

安装 kubernets

更新 apt 源

1
2
3
4
5
6
7
8
9

sudo apt-get update && sudo apt-get install -y apt-transport-https curl

sudo curl -s https://mirrors.aliyun.com/kubernetes/apt/doc/apt-key.gpg | sudo apt-key add -

sudo tee /etc/apt/sources.list.d/kubernetes.list <<-'EOF'
deb https://mirrors.aliyun.com/kubernetes/apt kubernetes-xenial main
EOF

sudo apt-get update

安装 kubectl kubeadm kubelet

在 master 和 node 上都安装

1
2

sudo apt-get install -y kubelet=1.17.3-00 kubeadm=1.17.3-00 kubectl=1.17.3-00
sudo apt-mark hold kubelet=1.17.3-00 kubeadm=1.17.3-00 kubectl=1.17.3-00

启动 kubelet

1

sudo systemctl enable kubelet && sudo systemctl start kubelet
